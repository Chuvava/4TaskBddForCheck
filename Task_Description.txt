  ОПИСАНИЕ ЗАДАНИЯ
Разработать автоматический тест по следующему сценарию:
1. Переход на сайт http://www.cars.com
Ожидаемый результат:
Открывается главная страница сайта

2. В блоке меню выбираем "Research"
Ожидаемый результат:
Открывается страница с выбором авто для поиска

3. Выполняем поиск по случайно выбранным характеристикам
Ожидаемый результат:
Значения успешно выбраны в combobox.
После нажатия на кнопку Search открывается страница с описанием авто

4. На вкладке Trims переходим по ссылке "[Наименование модели] trim comparison"
Ожидаемый результат: 	
Открылась страница с характеристиками выбранной модификации

5. Запоминаем характеристики авто для базового(первого) трима последующего сравнения
(характеристики: Available Engines, Transmissions)
Ожидаемый результат:
Характеристики сохранены

6. Переходим на главную страницу http://www.cars.com
Ожидаемый результат:
Открывается главная страница сайта

7. Выполняем шаги 2-5 для еще одного набора данных для поиска
Ожидаемый результат:
Поиск и отображение информации об авто успешно производится; характеристики сохранены

8. Через меню Buy переходим по ссылке "Find cars for sale"
Ожидаемый результат:
Открылась страница для поиска предложений по покупке (https://www.cars.com/shop/)

9. В разделе Side-by-Side Comparisons переходим по кнопке (ссылке) Compare cars	
Ожидаемый результат:
Открылась страница для выбора авто для сравнения

10. Выбираем модель, отобранную на шагах 2-5 и переходим по кнопке (ссылке) Start compare now
Ожидаемый результат:	
Открылась страница c выбранным авто для сравнения

11. Используя Add another car добавляем к сравнению модель, полученную на шаге 7	
Ожидаемый результат:
Модели успешно выбраны для сравнения

12. Проверяем страницу сравнения 2-ух моделей
Ожидаемый результат:	
Характеристики авто на странице соответствуют тем, что получены на шагах 2-7
